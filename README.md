# Guía para organizar materiales de una tesina

Plantilla lista para copiar y usar en tu proyecto. Incluye una estructura clara de carpetas desde el inicio hasta la finalización del proyecto.

## Uso

1. **Copia** la carpeta `Tesina/` completa a la ubicación donde quieras trabajar (por ejemplo, tu carpeta de proyecto o OneDrive).
2. **Empieza** a guardar tus archivos en las carpetas correspondientes.

## Formatos de documento

Puedes redactar la tesina en:
- **Word:** archivos `.docx` (plantillas en .txt para copiar a Word; capítulos con nombre tipo Capítulo1_Introducción).
- **LaTeX:** documento principal `main.tex` y capítulos `capitulo1_introduccion.tex`, `capitulo2_metodologia.tex` en `01_Borradores_y_textos_propios`. Compila con `pdflatex main` o `latexmk -pdf main`. Las versiones entregadas pueden ser `.pdf`.

## Estructura de carpetas

| Carpeta | Contenido |
|--------|-----------|
| **01_Borradores_y_textos_propios** | Capítulos, notas y textos que tú escribes |
| **02_Referencias_bibliográficas** | Libros, Artículos, Tesis, Referencias (gestor Zotero/EndNote) |
| **03_Datos** | Datos originales, procesados, Análisis (scripts y resultados), Resultados |
| **04_Material_adicional** | Imágenes, presentaciones, correspondencia con tutores |
| **05_Versiones_entregadas** | Copias de cada entrega (v1, v2 con correcciones, etc.) |

## Consejos prácticos

### Nomenclatura
- **Archivos y carpetas:** Usa un criterio consistente, por ejemplo:
  - `Año_Mes_Día_Tema` (ej. `2025_03_15_Reunion_tutor`)
  - `CapítuloX_Tema` (ej. `Capítulo3_Resultados`)
- **Referencias:** Incluye siempre **Autor_Año_Título** en el nombre del archivo (ej. `Garcia_2020_Introduccion_estadistica.pdf`).

### Copias de seguridad
- Haz copias de seguridad con regularidad (nube y/o disco externo).
- No dependas de un solo dispositivo.

### Referencias
- Organiza por tipo: Libros, Artículos, Tesis, Referencias.
- Usa un **gestor de referencias** (Zotero, Mendeley, EndNote) para citas y bibliografía.
- Mantén los PDFs con nombre descriptivo (autor, año, título).

### Mantenimiento
- Revisa y depura carpetas cada cierto tiempo.
- Elimina duplicados y material obsoleto.

### Separación clara
- Mantén tus **textos originales** (borradores) separados del **material de consulta** para evitar confusiones.

## Importancia de la organización

Un sistema ordenado te permite:
- **Localizar** información con rapidez.
- **Reducir el riesgo** de perder materiales importantes.
- **Acelerar** la redacción y las revisiones.

Aplicando esta metodología tendrás una base sólida para avanzar con tu tesina de forma ordenada y eficiente.
